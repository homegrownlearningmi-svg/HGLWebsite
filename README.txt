HOW SWAPPING WORKS
Replace any illustration or pattern file in GitHub under the exact same
path and filename. The HTML does not need to change. GitHub Pages will deploy
the replacement automatically. Use Ctrl+Shift+R if your browser shows a cached
copy.

PROFILE PHOTO
The package includes the current embedded photo as prof.png so the site works
immediately. Replace assets/illustrations/prof.png whenever the new portrait is
ready.
